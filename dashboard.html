Tab Assistant

API设置|LLM API

支持官方与私有化部署的 Firecrawl，当前仅提供界面配置。

导入类型

控制通过右键菜单导入标签页后的行为。

提示：仅对“导入当前标签页”和“导入所有标签页”生效。

抓取设置

选择网页抓取解析 Markdown 的策略。

提示：本地抓取时，请在解析完成前保持标签页处于打开状态。

AI 笔记设置

选择默认笔记风格，并为模型提供额外补充信息。

补充信息会一并传给模型，用于强调关键点或额外格式要求。

Hub 导入

选择 Hub 中导入 JSON 时的合并策略。

提示：Hub 导入/导出用于批量 JSON 文件，不影响“导入当前/所有标签页”的网页采集功能。

通用配置

配置抓取与 AI 生成的批量参数，保存后立即生效（超时、抓取/AI 并发数、重试次数、一键生成）。

提示：勾选后点击 AI 按钮将自动抓取并生成笔记。
